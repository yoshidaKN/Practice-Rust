## Rustとは？
- RustはMozillaが開発したシステムプログラミング言語であり、コンパイル言語である
- 安全性に特化
    - 借用や所有権という概念で、安全でクラッシュすることのない
    - RustはC/C++と同様の速度
- Webバックエンド / フロントエンドの開発に最適
    - WebAssemblyにとって最良の選択
- やること
    - DIESELとRocketを使って簡単なWebAPIを作成する
